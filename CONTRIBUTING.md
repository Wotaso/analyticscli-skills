# Contributing

This repository contains the published AnalyticsCLI skill pack.

## Before You Change Anything

- Open an issue first for larger changes.
- Prefer focused fixes over broad repo-wide refactors.

## Good Contributions

- broken install instructions
- unclear version support guidance
- missing examples or references
- compatibility notes for Codex, Claude Code, or OpenClaw

## Less Useful Contributions

- repo-wide refactors without a matching product change
- formatting-only churn
- changes that make a skill broader instead of more focused

## Skill Design Rules

- Keep one skill focused on one job-to-be-done.
- Keep `SKILL.md` compact and move details into `references/`.
- Version the instruction pack separately from the target SDK or CLI.
- Split into sibling skills when major-version guidance diverges.
